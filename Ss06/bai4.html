5. Groups of selector

A comma-separated list of selectors represents the union of all elements selected by each of the individual selectors in the list. (A comma is U+002C.) For example, in CSS when several selectors share the same declarations, they may be grouped into a comma-separated list. While space may appear before and/or after the comma.

CSS example:

In this example, we condense three rules with identical declarations into one. Thus,

h1 { font-family: san-serif }
h2 { font-family: san-serif }
h3 { font-family: san-serif }

is requavalent to:

h1, h2, h3 { font-family: san-serif }
